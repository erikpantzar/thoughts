Stats

We got a soccer team wich I would like to keep track of all us players, and setup a base and profile for each player based on my perception of theirs and mine ability on the field.
For example I like to sprint past opponents and maybe do some Ronaldo signature moves, but I can only sprint a limited amount due to my stamina being poor. So my player stat would be something like this: 

var erik = new Player{
    position: position[4], // attacker
    speed: 80,
    technique: 60,
    stamina: 40,
    // array
    specialty: speciality[2], speciality[3], speciality[4]
};

Also there would be some string properties like name, position, and profile image or something.

Reusability

The most advanced thing I’ve thought of is to have some properties collect their values from arrays; In FIFA you can have certain abilites such as “Sprinter”, “Longshot taker”, “Injury prone” or “Engine”. My way of getting this is to set the value to something out of an array to be able to reuse those properties on other player without worrying about misstyping the value and keep it expandable.
